# Ipython

> IPython是一种基于Python的交互式解释器，提供了强大的编辑和交互功能



## 1.? 打印IPython简介



## 2.object？内省功能

- 变量后加上`?`，可以打印该变量的详细信息
- 如果加`??`,则还可以查看函数或模块对象的源代码



## 3.tab 自动补全

* 输入命令时，按住tab键可查看可补全的内容



## 4.! shell_command 执行shell命令

```shell
eg：！ping www.baidu.com
```



## 5.%magic 显示所有魔术命令



## 6.%run 运行脚本

* py文件可通过%run命令当作python程序来执行，输入`%run 路径+文件名`即可



## 7.%pwd 显示工作路径



## 8.%ls 显示目录内容
