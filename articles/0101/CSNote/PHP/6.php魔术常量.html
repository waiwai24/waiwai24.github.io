PHP魔术常量

[TOC]

一. 基本概念

PHP 向它运行的任何脚本提供了大量的预定义常量。

不过很多常量都是由不同的扩展库定义的，只有在加载了这些扩展库时才会出现，或者动态加载后，或者在编译时已经包括进去了。

有八个魔术常量它们的值随着它们在代码中的位置改变而改变,些特殊的常量不区分大小写

二. _LINE_

显示文件中的当前行号

<?php
echo '这是第 " '  . __LINE__ . ' " 行';
?>

out：这是第 “ 2 ” 行

三. _FILE_

文件的完整路径和文件名。如果用在被包含文件中，则返回被包含的文件名

<?php
echo '该文件位于 " '  . __FILE__ . ' " ';
?>

四. _DIR_

文件所在的目录。如果用在被包括文件中，则返回被包括的文件所在的目录

<?php
echo '该文件位于 " '  . __DIR__ . ' " ';
?>

五. _FUNCTION_

返回该函数被定义时的名字

<?php
function test() {
    echo  '函数名为：' . __FUNCTION__ ;
}
test();
?>

六. _CLASS_

回该类被定义时的名字

<?php
class test {
    function _print() {
        echo '类名为：'  . __CLASS__ . "<br>";
        echo  '函数名为：' . __FUNCTION__ ;
    }
}
$t = new test();
$t->_print();
?>

七. _TRAIT_

代码复用的一个方法，称为 traits

八. _METHOD_

返回该方法被定义时的名字

<?php
function test() {
    echo  '函数名为：' . __METHOD__ ;
}
test();
?>

九. _NAMESPACE_

当前命名空间的名称

<?php
namespace MyProject;
 
echo '命名空间为："', __NAMESPACE__, '"'; // 输出 "MyProject"
?>
