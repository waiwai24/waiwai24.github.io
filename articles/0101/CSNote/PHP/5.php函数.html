PHP真正的威力来源于他的函数，在PHP中，提供了超过1000个内建的函数

一. 创建函数

PHP 函数准则：

  • 函数的名称应该提示出它的功能
  • 函数名称以字母或下划线开头（不能以数字开头）
<?php
function functionName($参数)
{
    // 要执行的代码
}
?>

返回值：使用return语句

二. 变量函数

将一个变量作为函数名来调用函数‘

<?php
class Foo
{
    function Variable()
    {
        $name = 'Bar';
        $this->$name(); // 调用 Bar() 方法
    }

    function Bar()
    {
        echo "This is Bar";
    }
}

$foo = new Foo();
$funcname = "Variable";
$foo->$funcname();  // 调用 $foo->Variable()

?>
