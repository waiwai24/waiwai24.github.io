表格

Markdown 表格由 「竖线 |」、「减号 -」、「冒号 :」三种符号组成。

  • 竖线 用来定义列，每两个竖线之间为一个单元格元素；
  • 减号 用来定义分割线，也就是分割表头和数据体；
  • 冒号 配合减号使用，用于定义列数据的对齐属性。
|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

展示效果：

表头 表头
单元格 单元格
单元格 单元格

对齐方式:

我们可以设置表格的对齐方式：

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

展示效果：

左对齐 右对齐 居中对齐
单元格 单元格 单元格
单元格 单元格 单元格
